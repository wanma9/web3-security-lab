# web3-security-lab
Laboratorio Online de Seguridad Web3
Descripción

Este repositorio contiene un laboratorio práctico orientado al aprendizaje de conceptos básicos de seguridad en Web3, utilizando contratos inteligentes de Ethereum con fines educativos.

El entorno está diseñado para ser reproducible y completamente online, sin uso de sistemas reales ni datos sensibles.

Objetivo

El proyecto tiene como finalidad crear un entorno que permita:

Instalar herramientas del ecosistema Ethereum/Solidity
Analizar contratos inteligentes vulnerables de ejemplo
Automatizar la ejecución de análisis
Registrar resultados y logs
Almacenar información en una base de datos
Documentar el proceso para su reproducción
Uso básico
Clonar el repositorio:
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
Instalar las herramientas necesarias siguiendo la documentación en /docs.
Ejecutar los scripts de análisis:
bash scripts/run_analysis.sh
Revisar los resultados en la carpeta /logs.
